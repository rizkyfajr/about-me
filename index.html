e-KFPB

Mengembangkan Aplikasi Internal KFPB dan beberapa modul didalamnya. digunakan untuk mengelola data internal perusahaan di Kimia Farma. Aplikasi ini memiliki beberapa modul yang mendukung berbagai bagian di Kimia Farma, seperti SPPTEK (Surat Permintaan Pembelian Barang Teknik) yang digunakan oleh bagian Teknik untuk permintaan perbaikan di lingkungan Kimia Farma, serta QMS (Quality Management System) untuk pengendalian dokumen, Change Control, dan Penyimpangan. Aplikasi ini dikembangkan menggunakan Native PHP dan SQL.

eSDM

Aplikasi eSDM digunakan untuk mendukung absensi karyawan di Kimia Farma. Aplikasi ini dikembangkan menggunakan PHP dan SQL sebagai database. Selain itu, eSDM juga dilengkapi dengan API yang memungkinkan pengiriman data pegawai dan absensi ke aplikasi Kimia Farma Pusat. Aplikasi ini memiliki dua level pengguna, yaitu admin dan pegawai sebagai pengguna biasa.
